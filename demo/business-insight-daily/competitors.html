友商深度洞察

五大云厂商最新战略动向 · 每条分析均含深度研判与可溯源数据来源

● Amazon AWS
新品密集
Bedrock AgentCore 上线自主支付能力 战略级
AWS 在 2026 年 5 月正式推出 Bedrock AgentCore 的自主支付能力，标志着 AI Agent 从"对话交互"迈向"交易闭环"的关键一步。Agent 现在可以自主完成从用户意图识别→商品选择→下单→支付→订单确认的完整电商交易链路。IAM 审计和安全策略确保了企业级交易的合规性，这是在 AWS 原有 MCP Server 沙箱化 Python 执行与 IAM 审计基础上的重要功能延伸。AWS 正在将 AI Agent 从"辅助工具"进化为"自主执行者"，这对电商、 SaaS 订阅管理、自动化采购等场景具有深远影响——企业可以将重复性交易流程完全交给 Agent，释放人力处理更高价值的决策工作。
来源：AWS Machine Learning Blog →
SageMaker AI Agentic 模型定制 战术级
AWS SageMaker AI 推出 Agentic 模型定制能力，将 AI 模型的部署周期从数月缩短至数天。这一能力允许企业针对特定业务场景快速微调 Agent 行为，而不需要从头训练。结合 Bedrock 的 Agent 编排能力，AWS 正在构建从"模型训练→Agent 定制→生产部署"的一站式平台。相比 Azure 和阿里云，AWS 的核心优势在于 IAM 权限模型的成熟度和企业级安全合规能力——这对金融、医疗等强监管行业至关重要。
来源：AWS Blog →
官方来源：AWS ML BlogAWS Official Blog
● Microsoft Azure
战略升级
Copilot Studio MCP 连接器上线 战术级
Microsoft 在 2026 年 5 月正式推出 Copilot Studio 的 MCP（Model Context Protocol）连接器，允许企业将自定义 MCP Server 接入 Copilot 生态。这一举措使 Copilot 从"Microsoft 365 内的 AI 助手"进化为"企业全系统的 AI 入口"。Azure 的策略非常清晰：依托 Office 365 + Dynamics 365 + GitHub 的庞大生态，将 Copilot 打造为企业 AI 的唯一入口。MCP 连接器的上线意味着企业现有的 CRM、ERP、HR 系统都可以通过标准协议与 Copilot 交互，而不需要定制开发。这与 AWS 的"开放 Agent 平台"策略形成鲜明对比——Azure 更强调"生态锁定"，而 AWS 更强调"开放标准"。从市场反应看，已有超过 200 家企业接入 Copilot Studio MCP 连接器预览版。
来源：Microsoft Tech Community →
Azure AI Agent Service GA 战略级
Azure AI Agent Service 正式全面上线（GA），标志着 Azure 在企业级 Agent 编排领域的布局完成。该服务支持多 Agent 协作、RAG 集成、安全合规审计等企业级特性，并与 Microsoft Fabric 数据分析平台深度集成。Azure 的策略是"数据+AI+分析"一体化——Fabric 负责数据层，AI Agent 负责智能层，Power BI 负责展示层。相比阿里云和 AWS，Azure 在企业级数据分析与 AI 的结合上最为深入。
来源：Azure Blog →
官方来源：Microsoft Tech CommunityAzure Blog
● 阿里云
重大战略
全栈 Agent 化升级 + 真武 M890 芯片 战略级
阿里云在 2026 年 5 月 20 日云峰会上发布了完整的 Agent 全景图，宣布从基础设施到模型到推理的"全栈 Agent 化"战略。核心发布包括：① 真武 M890 AI 芯片正式官宣——这是阿里云首款自研 AI 推理芯片，对标 NVIDIA 的中端推理市场；② Qwen3.7-Max 模型发布，在多项基准测试中达到业界领先水平；③ 百炼平台全面 Agent 化——从单一模型服务平台升级为 Agent 开发、部署、运维的全生命周期平台。阿里云的战略逻辑是"芯片→云计算→大模型→Agent 应用"的垂直整合，这与 AWS 和 Azure 的"平台开放"路线不同。真武 M890 芯片的加入，使阿里云成为全球少数几家拥有"芯片+云+模型+Agent"全栈能力的云厂商。但与 NVIDIA B300（288GB HBM3e）相比，真武 M890 在高端推理市场仍存在差距。
来源：阿里云官网 →
百炼平台开放竞品模型上架 战略级
阿里云百炼平台宣布开放上架 GLM-5.1、KimiK2.6 等竞品模型，这是中国云厂商首次在模型平台上"拥抱"竞争对手。这一策略转变意义重大：从"自研模型垄断"到"生态开放共赢"的转型。对于中长尾开发者而言，百炼平台成为"模型超市"—可以在一个平台上使用多家模型，降低了切换成本。这也是应对火山引擎 AgentPlan 订阅制（40-200元/月）和腾讯云 Hy3 开源策略的竞争反应。
来源：阿里云官网 →
官方来源：阿里云官网百炼平台
● 火山引擎
模式创新
AgentPlan 订阅服务上线 DeepSeek V4 战略级
火山引擎推出 AgentPlan 订阅服务，以 40-200 元/月的固定费用提供 DeepSeek V4 系列模型的调用能力。相比传统的按 Token 计费模式，AgentPlan 最高可为企业节省 80% 的模型调用成本。这一模式的创新之处在于：① 将 AI 模型从"按量付费"转向"订阅制+按量混合"——这是 SaaS 行业已验证的最优变现模型；② 订阅制锁定的是"Agent 消费场景"而非"模型调用量"，意味着火山引擎正在从 IaaS/PaaS 的计费逻辑转向 SaaS 的 LTV 逻辑；③ 固定月费大幅降低了中长尾客户的 AI 采用决策门槛。火山引擎的 AgentPlan 与字节跳动的豆包大模型生态深度绑定，形成了"模型+平台+订阅"的三层商业闭环。
来源：火山引擎官网 →
豆包大模型搭载 700 万+ 辆车 战术级
字节跳动旗下豆包大模型已搭载超过 700 万辆汽车，日均交互次数达 3000 万次。AI 座舱正在成为汽车行业差异化竞争的核心——从简单的语音助手升级为"车载 Agent"。火山引擎在汽车 AI 领域的布局领先于其他云厂商，700 万辆的搭载量意味着其车载 AI 产品已经跨越了"早期市场"进入"主流市场"阶段。
来源：火山引擎官网 →
官方来源：火山引擎官网火山方舟
● 腾讯云
产品加速
WorkBuddy 首日算力十连扩 战略级
腾讯云 AI 效率智能体 WorkBuddy 上线首日用户大量涌入，算力需求超出预期 10 倍，腾讯云紧急进行十连扩。WorkBuddy 是中国市场目前覆盖面最广的 AI 效率智能体，集成了腾讯会议、企业微信、腾讯文档、腾讯邮箱等全系办公产品的 AI 能力。WorkBuddy 的核心竞争力在于"微信+企业微信"的社交生态协同——超过 12 亿微信用户为 WorkBuddy 提供了天然的分发渠道。这是 AWS、Azure、阿里云都不具备的独特优势。
来源：腾讯云官网 →
Hy3 preview 登顶 OpenRouter 战略级
腾讯混元大模型 Hy3 preview 连续多日在 OpenRouter 模型排行榜上位居第一，标志着腾讯在大模型能力上已达到全球第一梯队。同时，腾讯云 Q1 To B 营收达到 599 亿元，同比增长 14%。Hy3 的开源策略（Hy3 在 GitHub 上开源）使其在开发者社区获得了广泛认可。腾讯云的"模型开源+Agent 平台开放+社交生态协同"的三位一体策略，正在形成独特的竞争壁垒。
来源：腾讯云官网 →
